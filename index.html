Um programador júnior apaixonado por criar soluções digitais envolvendo HTML, CSS e JavaScript. Se você procura um programador junior com potencial, entusiasmo e habilidades técnicas sólidas, entre em contato comigo para discutir como posso contribuir para o seu time.
